# P23 - Restore the required MIT copyright notice

A commit on this branch removed the existing copyright notice while retaining an
MIT license whose own condition requires the copyright and permission notice be
included in copies or substantial portions of the software. Distributing the
repository or extension after this change would omit the attribution required by
the project's existing license. Retain the `Copyright (c) 2024 Simon Benedict`
line. Source: `LICENSE:3`.
